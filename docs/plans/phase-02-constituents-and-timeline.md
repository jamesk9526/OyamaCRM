# Phase 02 — Constituents and Timeline

## Goal

Finish the missing stewardship and data-quality features around constituent records.

## Already in place

- Constituents list / create / edit / detail flows
- Household panel and create/update timeline writes
- Search and core filters

## Remaining scope

- Soft credits / influencer links
- Custom fields
- Tag management and bulk updates
- Saved segments
- CSV import / dedupe
- Bulk edit
- Wealth indicators

## Remaining implementation steps

1. Add soft-credit and relationship modeling beyond households.
2. Add configurable custom fields to constituent profiles.
3. Build tag CRUD plus bulk assignment/removal.
4. Build saved segments and smart-list foundations.
5. Finish import, dedupe, and bulk-edit workflows.
6. Surface wealth-screening / capacity indicators where relevant.

## Exit criteria

- Staff can manage constituent metadata and relationships beyond the core profile.
- Import and bulk-edit flows reduce manual cleanup work.
- Segmentation and tagging support real stewardship operations.
