# OyamaCRM User Guide

Last updated: 2026-05-14

This guide is for nonprofit staff using OyamaCRM day to day.

## Getting Started

1. Open the Dashboard.
2. Use the Start Here cards to choose your first action.
3. Check Today's Focus for urgent work.
4. Open the top-right notifications bell to review unread alerts.
5. Use global search in the top bar to jump to donors, tasks, campaigns, and tools.

## Dashboard Overview

The dashboard answers three questions:

- What should I work on now?
- What needs attention today?
- Where should I go next?

Use:

- Start Here: common actions like adding a donor or recording a donation.
- Today's Focus: open tasks, overdue tasks, recent giving, campaign load, and steward signals.
- Widget grid: deeper trend and workload detail.

## Adding Donors

1. Go to Constituents.
2. Create a new donor profile.
3. Enter contact details and communication preferences.
4. Save, then open the donor record for follow-up actions.

## Recording Gifts

1. Go to Donations and choose new donation entry.
2. Select the donor.
3. Enter amount, date, payment method, and campaign if applicable.
4. Save and confirm acknowledgment follow-up.

## Creating Tasks

1. Open Tasks.
2. Create a task with a clear title and due date.
3. Assign the task to the right staff member.
4. Link the related donor, campaign, or donation when relevant.

## Using the Calendar and Follow-Ups

1. Open Tasks or Appointments (module-dependent).
2. Filter by overdue, due today, or assigned to me.
3. Update schedule and complete tasks as work is done.

## Sending Communications

1. Open Communications.
2. Start from New Campaign or Start Here.
3. Choose audience and content preset.
4. Review, test, then schedule or send.

## Generating Letters

1. Open Letters and Printables.
2. Choose template.
3. Choose recipients.
4. Preview merge output.
5. Generate and move to print, mail, or email draft workflow.

## Using Steward

Steward provides suggestions and drafts.

- Always review suggestions before taking action.
- Use suggestions as guidance, not source of truth.
- Confirm any write action before completion.

## Managing Notifications

Use the top bar bell to:

- Mark read
- Snooze
- Dismiss
- Open related records

Prioritize high and medium alerts first.

## Running Reports

1. Open Reports.
2. Choose report module and date scope.
3. Apply filters.
4. Export only after verifying scope and totals.

## Importing Data

1. Open Data Tools Import.
2. Upload CSV.
3. Map fields.
4. Validate required fields.
5. Review duplicates.
6. Confirm import.

## Troubleshooting

- If data looks stale, use page refresh controls.
- If saving fails, check required fields and try again.
- If access is blocked, ask an admin to review your role permissions.
- Use Help Center from the top bar for route-specific guides.
