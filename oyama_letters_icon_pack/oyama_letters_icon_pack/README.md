# OyamaLetters Icon Pack

This pack contains 28 custom icon concepts for the OyamaLetters letter/document studio.

## Included formats

- `svg/` — editable transparent vector source files, 256x256 viewBox
- `png-512/` — transparent 512x512 PNG exports
- `oyama-letters-icons-preview.png` — quick contact sheet preview

## Style

- Primary green: #0B5D2A
- Dark green: #06411F
- Soft background: #EAF6EE
- Transparent icon backgrounds for app use

## Suggested usage

Use these for the OyamaLetters sidebar, workspace cards, ribbon actions, empty states, queue status cards, and Help articles. SVGs are the best choice for the web app; PNG files are included for quick mockups and documentation screenshots.
