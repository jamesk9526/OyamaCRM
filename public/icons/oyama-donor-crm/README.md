# OyamaCRM Donor Icon Pack

This pack contains **124 custom icons** for the OyamaCRM DonorCRM workspace.

## Included formats

- `icons/svg/` — editable vector source files
- `icons/png/512/` — 512 × 512 transparent PNG exports
- `preview/oyama-donor-crm-icons-preview.png` — visual preview sheet
- `manifest.json` — file list and categories

## Style

- Clean nonprofit CRM style
- Oyama green outline: `#166534`
- Light green fills: `#dcfce7`, `#f0fdf4`
- Rounded strokes and soft shape language
- Designed for sidebars, cards, empty states, buttons, docs, and design mockups
