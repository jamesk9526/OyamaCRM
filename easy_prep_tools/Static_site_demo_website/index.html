Architecture

Production-grade from the ground up

⚡

Next.js 16 + React 19

App Router with Server Components, TypeScript end-to-end, and a modular CRM workspace architecture.

🔒

JWT + RBAC + Watchdog

In-memory access tokens, secure refresh cookies, role-aware permissions per module, and AES-256-GCM encrypted vault secrets.

🗃

MySQL + Prisma ORM

Transactional data in MySQL via Prisma. External Watchdog store for security events. Full backup/restore controls built in.

🔠

QuickBooks Integration

Native OAuth QuickBooks Online sync queue. Manual push workflow with immutable audit trail. Rate-limited, duplicate-safe.

🤖

Steward AI

AI-powered work signal engine surfacing donor follow-ups, task prioritization, and relationship insights.

🚀

PM2 + Express API

Express 5 API layer at /api with PM2 process management. Import/export tooling and Vitest smoke coverage.
