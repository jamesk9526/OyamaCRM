Capabilities

A donor-first platform that still covers the whole organization

This pass is centered on DonorCRM: fundraising operations, stewardship execution, reporting, and communications, with linked client-services and event workspaces.

DonorCRM

Fundraising command center

Revenue progress, donor retention, campaign tracking, top donors, and stewardship actions live together on the main dashboard.

  • YTD revenue and goal progress
  • Donor retention year-over-year
  • Campaign performance and grants visibility
  • Workspace widgets that staff can rearrange
Constituents

Real constituent records

Constituent list, statuses, YTD giving, engagement scores, and tags are visible without forcing staff into a maze of subpages.

Reports

Reporting that staff can actually use

Template library, report manager, and a ribbon-based report viewer with multiple chart modes, export, filtering, and print preview.

Stewardship

AI-guided donor engagement, not generic chat

Steward Signals turns donor data into actionable queues: high-opportunity donors, at-risk relationships, monthly giving candidates, and daily guidance.

  • Daily Steward Thought with clear reasoning
  • Opportunity counts and at-risk detection
  • Growth ideas with confidence scoring
  • Draft-first workflow support for next steps

Platform Modules

Three CRM workspaces plus platform tools

DonorCRM

Constituents, donations, campaigns, grants, tasks, communications, letters, reports, and stewardship AI.

Compassion CRM

Privacy-first client services, appointments, cases, follow-ups, and scoped records.

Events CRM

Event-first operations with guests, tickets, tables, check-in, sponsors, and event reports.

Watchdog

Security feed, encrypted vault, backup tools, and cross-module operational visibility.

Desktop App

A lightweight shell for offices that want OyamaCRM to launch like a native desktop workspace.

Easy Desktop Application

Open OyamaCRM like a native desktop workspace

The desktop shell gives front-desk and office teams a cleaner launch experience while keeping the same CRM routes, auth flow, and module access.

What the desktop app adds

  • Step-by-step onboarding that binds your CRM URL in under a minute
  • A clean desktop frame with one-click launch behavior for staff
  • Simple appearance controls to match office preferences
  • Same secure login and full module access as the web experience

Ideal for front-desk teams and offices that want a consistent, app-like CRM launch experience without browser clutter.

OyamaCRM

  • Free forever for self-hosted nonprofits

Architecture

Production-grade from the ground up

Next.js + React

Modern app-router foundations with modular workspaces and shared product chrome.

JWT + RBAC + Watchdog

Role-aware security, auditing, and visibility across modules.

MySQL + Prisma ORM

Transactional data with practical operational tooling around it.

QuickBooks Integration

Native sync queue with manual controls and audit safety.

Steward AI

AI-powered work signals, guidance, and relationship support.

Express API Layer

Import/export workflows, background jobs, and integration-ready endpoints.
